Styles

The syntax of the style elements and attributes is tagname style="property:value;"

The following are the properties to choose from:

  1. Background-color for background color
  2. Color for text colors
  3. Font-family for text fonts
  4. Font-size for text sizes
  5. Text-align for text alignment
