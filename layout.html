There are 4 major ways to implement a webpage layout in HTML, all of them are based on CSS. They are:

  • CSS Framework - This gives you pre-made templates from using frameworks like W3.CSS or bootstrap
  • CSS Grid - This is a grid-based layout system, with rows and columns, making it unnecessary to use floats and positioning.
  • CSS Float Layout
  • CSS Flexbox

The following are the elements of any HTML page

  1. Section - A group of contents usually having the same theme/heading
  2. Article - This is independant content. That means it should make sense in itself and you can place it anywhere on the webpage. You use this when you're writing blog posts or newspaper articles
  3. Header - Introductory content like the heading using h1-h6 and/or a logo
  4. Footer - This can have many compotents like author, contact info, back to the top links, etc
  5. nav - This is the navigation bar. Usually you would like to put it right below the header. This does not necessarily have to have all the navigation links on the page
  6. Aside - This defines some content on the side, like a sidebar
